# Charlotte Run Clubs (Mobile Web)

Mobile-first run club listing app that scrapes from:

- https://www.charlotterunningclub.org/Run-Clubs

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Features

- Live scrape of Charlotte Running Club page via `/api/run-clubs`
- Search by club, day, time, or location
- Day dropdown filter
- Mobile UI styled like iOS list screen
